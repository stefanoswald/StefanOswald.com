Night 198

The Sprinkler Run

Month 7: Outside!

The Hendersons' sprinkler is on!

Chik-chik-chik-whoosh...

The big kids leap the middle.

Olivia runs the edge.

Her edge. The gentle end.

Run in, mist!

Cold! Giggle! Run out!

Run in, mist! Run out!

The rainbow lives in the spray.

There it is! See it?

Run through the rainbow's foot!

Wet curls. Wet everything.

The towel waits on the fence.

The summer waits nowhere.

It is all here, all now,

chik-chik-chik-whoosh,

every golden afternoon.
