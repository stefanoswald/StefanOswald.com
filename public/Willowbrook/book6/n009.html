Night 9

The Willowhawk Gets Wheels

Month 1: How Things Work

Build week three: Wheels. (Donors: one retired wagon, one sacrificial scooter. The Willowhawk rolls before it flies..." Everything walks before it flies. "by the workshop chalkboard, which is developing a philosophy section.) The rolling tests, driveway, detail person timing: Run one: veer left.

Run four: straight, slow. Run seven: straight, fast, Mateo whooping, bedsheet wings flexing like they're dreaming about their real job... And one small spectator at the fence line, vibrating: "When the flying? When the hill, Theo.

Kite hill, end of summer. The Willowhawk earns the hill one system at a time, wheels, then steering, then wings, then the day." (The workshop methodology: hers now too. One system at a time. The one-step rule, dressed in sawdust.)
