Night 269

The Wave Code Expansion

Month 9: Friends Old and New

The wave code underwent its constitutional convention this week, because Noor joined the window schedule, and three users need real setup. The expanded vocabulary, agreed at the bench and documented. Documented, because write things down, in all three notebooks: "One wave: hi. Two: good thing, tell at recess.

Flat palm: hard morning, bring the quiet. Spinning finger: unbelievable thing happened, run at recess. Both hands up: I got a answer to a question one of us had. Fist on heart: the biggest one.

Means: I know today's heavy for you specific. I'm at the glass. (Used sparingly. The sacred ones stay sacred by staying rare, the battery rule, standing.)" And the document's final part, hers: "This code passes down.

When we go up a grade, the windows stay. The code gets taught to whoever lands at these desks. Signed: O, W, N." (It passed down. A quick check confirmed the windows years later: children it never met, waving the code, fist on heart on the heavy days. Setup outlives its engineers. The good kind always does.)
